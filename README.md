# vscode-inf
**Windows INF** File support for Visual Studio Code, Generated using [Yeoman (yo)](https://yeoman.io/) and vscode generator

### How to install
1. Open Visual Studio Code
2. Press Ctrl+Shift+P
3. Write 'VSIX'
4. Select 'Extensions: Install from VSIX...'
